COMPETENCES

Html

HTML (HyperText Markup Language) est le langage standard pour structurer le contenu des pages web. Il utilise des balises pour organiser des éléments comme titres, paragraphes, images et liens.

Voir le cours

Css

CSS (Cascading Style Sheets) est un langage utilisé pour décrire l'apparence et la mise en page des pages web. Il permet de définir les styles des éléments HTML comme couleurs, polices, espacements et disposition.

Voir le cours

java-script

JavaScript est un langage de programmation utilisé pour rendre les pages web interactives. Il permet de manipuler le contenu HTML et CSS, gérer les événements, et créer des fonctionnalités dynamiques comme animations et formulaires interactifs..

Voir le cours

Bootstrap

Bootstrap est un framework CSS open-source pour faciliter le développement de sites web réactifs et mobiles. Il propose des classes CSS et des composants JavaScript pour typographie, formulaires, boutons, navigation, et autres éléments d'interface utilisateur.

Voir le cours

Php

PHP (Hypertext Preprocessor) est un langage de script côté serveur pour développer des pages web dynamiques. Il permet de générer du contenu HTML, interagir avec des bases de données, gérer les sessions et les cookies, et plus encore.

Voir le cours

MySQL

MySQL est un système de gestion de base de données relationnelle (SGBDR) open-source. Il utilise le langage SQL pour gérer et manipuler les données, couramment utilisé pour les applications web en raison de sa performance, fiabilité et simplicité.

Voir le cours
